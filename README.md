# AI Resume Analyzer

An AI-powered resume analysis tool built using Python, Streamlit, and Google's Gemini API.

## Features

* Upload PDF resumes
* Extract resume content automatically
* Generate ATS-style scoring
* Identify strengths and weaknesses
* Suggest improvements
* Highlight missing skills

## Tech Stack

* Python
* Streamlit
* PyPDF
* Gemini API

## How It Works

1. Upload a resume PDF.
2. Text is extracted from the document.
3. Gemini analyzes the resume.
4. Results are displayed in a structured report.
